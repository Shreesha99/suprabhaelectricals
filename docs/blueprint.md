# **App Name**: Suprabha Electricals Hub

## Core Features:

- Landing Page: A single-page website featuring sections for Home, About, Services, Projects, and Contact Us, to effectively showcase Suprabha Electricals' offerings. Built with Next.js and the app router.
- Service Showcase: Interactive gallery displaying Suprabha Electricals' wide array of government electrical projects, focusing on the Karnataka region in India.
- Testimonials Section: Dedicated space to highlight client testimonials, building trust and credibility with prospective customers.
- AI Chatbot Integration: AI chatbot that can answer FAQs related to Suprabha Electricals’ services, working hours, contact information, and project history. The chatbot uses a tool to decide when to incorporate information from the business’ project portfolio in its response.
- Contact Form: Integrated contact form linked to suprabhaele@gmail.com.
- Project Display: Dedicated section showing all the government projects focused on Karnataka, India customers with pictures.
- Responsive Design: Ensure website responsiveness across various screen sizes (desktop, tablet, mobile) for optimal user experience.

## Style Guidelines:

- Primary color: Cherry Red (#E23D28) reflecting the company logo, conveying energy, and professionalism.
- Background color: Light beige (#F5F5DC), offering a soft contrast to the Cherry Red and creating a warm, inviting atmosphere.
- Accent color: A deep burgundy (#800020) to highlight key information and calls to action.
- Body font: 'PT Sans', a humanist sans-serif, will be used for the majority of the text on the site.
- Headline font: 'Playfair', a modern serif with a fashionable look, will be used to draw attention to headings. Note: currently only Google Fonts are supported.
- Simple and professional icons to represent services and navigation items.
- Clean and structured single-page layout with clear section divisions for easy navigation.
- Subtle scroll-triggered animations for smooth transitions between sections.
